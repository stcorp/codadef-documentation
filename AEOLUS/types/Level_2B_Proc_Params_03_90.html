4Report_Measurement_Center_DateTime
ascii uint8
If set to True the edge values reported for a wind result will be taken from the center datetime value of the first and last included measurement as reported in the L1B product. If set to False the first measurement will be shifted backward and the last measurement will be shifted forward by half the measurement time to ensure the reported timespan matches with the actual measurement time. This prevents reporting identical start and stop datetime values in case a wind is based on a single measurement.
