detection rule

This product definition is applicable if a product matches the following rule:

bytes(/,0,12) == "PRODUCT=\"AE_" and
bytes(/,17,10) == "ALD_U_N_2C" and
bytes(/,95,23) == "L2B/L2C IODD Iss. 01.32"

bytes(/,0,12) == "PRODUCT=\"AE_" and
bytes(/,17,10) == "ALD_U_N_2C" and
