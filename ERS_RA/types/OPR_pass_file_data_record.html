1MCD
binary recordsize: 4
This element is the Measurement Confidence Data, which regroups all the flags related to the 1-Hz altimeter measurement. 1-Hz altimeter quality flags are derived from the quality of 20-Hz ocean elementary measurements. A 1-Hz altimeter quality flag is set to "bad" if at least one of the corresponding elementary estimates is "bad" (i.e. out-of-range); otherwise it is set to "good". These engineering level flags may be considered as useful information for software validation, but their systematic use for editing data is to consider with care.
